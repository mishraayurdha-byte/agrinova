# 🌱 AgriNova AI v2.0

**Growing Tomorrow with Smart Farming**

AgriNova AI v2.0 is an AI-powered smart farming platform built with Flask, TensorFlow, and Scikit-learn. It helps farmers detect crop diseases, predict crop yield, optimize irrigation, monitor weather, analyze farm data, and generate professional reports.

---

## Features

- 🌿 AI Crop Disease Detection (TensorFlow)
- 🌾 Crop Yield Prediction (Scikit-learn)
- 💧 Smart Irrigation Recommendation
- ☀️ Live Weather Integration (OpenWeather API)
- 🤖 AI Farming Assistant Chatbot
- 📊 Interactive Analytics Dashboard
- 📄 PDF, Excel, and CSV Report Generation
- 📈 Chart.js Visualizations
- 📱 Responsive Bootstrap 5 UI
