# Disease Detection

## Purpose

Detect crop diseases from leaf images using a TensorFlow AI model.

## Steps

### Step 1

Open:

```text
Disease Detection
```

### Step 2

Click:

```text
Upload Image
```

### Step 3

Select a crop leaf image.

Supported Formats:

- JPG
- JPEG
- PNG

### Step 4

Click:

```text
Analyze
```

### Results

The system displays:

- Disease Name
- Confidence Score
- Recommendation
- Treatment Advice

Example:

```text
Disease:
Leaf Blight

Confidence:
97.2%

Recommendation:
Apply recommended fungicide treatment.
```

# Yield Prediction

## Purpose

Estimate crop yield using machine learning.

## Required Inputs

- Crop Type
- Area
- Rainfall
- Temperature
- Humidity
- Soil Type

### Example

```text
Crop: Rice

Area: 2 Acres

Rainfall: 120 mm

Temperature: 28°C

Humidity: 70%

Soil: Loamy
```

### Output

```text
Predicted Yield:
6.2 Tons

Accuracy:
95%
```

# Tips for Best Results

## Disease Detection

Use:

- High-resolution images
- Good lighting
- Clear leaf visibility

Avoid:

- Blurry images
- Dark images
- Multiple leaves in one image

---

## Yield Prediction

Provide accurate:

- Rainfall values
- Soil type
- Temperature
- Humidity

More accurate inputs produce better predictions.

---

## Smart Irrigation

Use actual sensor readings whenever possible.
